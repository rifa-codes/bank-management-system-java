 Bank Management System (Java)

Project Description

This project is a simple **Bank Management System** developed using Java.
It is a console-based application that simulates basic banking operations such as account creation, deposit, withdrawal, and balance checking.

The system helps users understand how banking functionalities can be implemented using programming concepts.

 Features

*  Create a bank account
*  Deposit money
*  Withdraw money
*  Check account balance
*  Prevent withdrawal if balance is insufficient
*  Beginner-friendly implementation

 Technologies Used

* Java
* Core Java concepts
* Scanner class for input handling

 Concepts Covered

* Classes and Objects
* Encapsulation
* Conditional Statements (`if-else`)
* Loops
* Methods / Functions
* User Input Handling
